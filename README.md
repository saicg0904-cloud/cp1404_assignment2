# CP1404 Assignment 2: Travel Tracker 2.0

## Project Overview
This is a Python-based travel tracker application that allows users to manage a list of places they want to visit.
It includes both a console version (refactored from Assignment 1) and a Kivy GUI version.

## Features
- **Console Version**: List places, add new places, mark places as visited, and get random place suggestions.
- **GUI Version**: Interactive Kivy interface to view, add, and mark places as visited.
- **Data Persistence**: All data is stored in `places.json` and automatically loaded/saved.
- **Unit Tests**: Comprehensive tests for `Place` and `PlaceCollection` classes.
