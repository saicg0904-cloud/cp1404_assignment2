## Project Description
This project is a travel tracker that allows users to store and manage places they want to visit.
It includes a console version and a Kivy GUI version.

## Features
- Load and save places from JSON
- Add new places
- Mark places as visited/unvisited
- Sort places by name, country, priority, visited
- GUI with colour-coded buttons
- Error handling for inputs
- Important place messages
