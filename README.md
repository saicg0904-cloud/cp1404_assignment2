Student: Qiuhao Wu

## Project Description
This project is a comprehensive travel tracking system that allows users to manage places they wish to visit.
It includes both a console-based interface and a graphical user interface built with Kivy.

## Features
- Load and save travel places from JSON file
- Add new places with validation
- Toggle visited/unvisited status
- Sort by name, country, priority, and visited status
- Colour-coded buttons for visited/unvisited places
- Full error handling for user inputs
- Important place notifications
- Clean and intuitive GUI layout
